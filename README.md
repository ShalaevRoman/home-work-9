# Simple Site

Pixel-perfect responsive implementation of the [Simple Site UK-EN](https://www.figma.com/design/0AJ044qDtJVGznIF4X0hLx/Simple-Site-UK-EN) Figma design.

## Technologies

- HTML5
- CSS3 (BEM, Custom Properties, Flexbox, Grid, Media Queries)
- Google Fonts — Roboto 400/500

## Structure

```
home-work-9/
├── index.html
├── styles.css
└── README.md
```

## Breakpoints

| Breakpoint | Width |
|---|---|
| Mobile | 375px+ |
| Tablet | 768px+ |
| Desktop | 1200px+ |

## Sections

- **Hero** — title, description, CTA button
- **About Me** — centered text block
- **Works** — 6 stats items with icons
- **How I Work** — description and video thumbnail
- **Contacts** — footer with name and copyright
